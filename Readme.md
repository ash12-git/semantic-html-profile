# Semantic HTML Personal Profile

This project is a simple personal profile webpage created using **semantic HTML5**
elements. The focus of this task is on proper structure, readability, and
accessibility rather than styling or advanced functionality.

## Objective
The objective of this task is to understand how semantic HTML elements help in
organizing web content clearly and meaningfully, making it easier for both
browsers and developers to interpret.

## What This Project Includes
- Proper HTML5 boilerplate written manually
- Semantic elements such as:
  - header
  - nav
  - main
  - section
  - article
  - footer
- Internal navigation using anchor links
- Clean and readable HTML structure

## Tools Used
- Notepad (for writing HTML)
- Web browser (for testing)
- GitHub (for submission)

## Internship Task
This project was completed as part of a **Web Development Internship** to
strengthen core HTML fundamentals and follow best practices used in real-world
projects.
